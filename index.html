What Rastro handles

Built for how sales commissions actually work.

Not how a generic SaaS tool thinks they work.

FR Sync

Automatically pulls rep, customer, and transaction data directly from your FieldRoutes account. No CSV exports.

Per-Rep Commission Rates

Configure a unique commission percentage for each rep. Changes apply retroactively or from a set date — your call.

Cancellation Clawbacks

When an account cancels, Rastro recalculates that rep's commission automatically. No manual adjustments.

Manager Overrides

Set manager override percentages that automatically roll up from rep-level sales. Multi-tier support included.

Backend Tracker

Reps see their current, past, and projected backend payouts — updated in real time as cancellations come in.

Housing Goal Logic

Handles the edge case where a rep earns housing but loses it after cancellations. Threshold tracking built in.
